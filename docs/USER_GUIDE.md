## Getting Started

### Logging In

1. Go to your admin panel (e.g., `yoursite.com/admin`)
2. Enter your email and password
3. Click "Login"

**Tip:** Bookmark this page on your phone for quick access!

### The Admin Dashboard

After logging in, you'll see three tabs at the bottom (on mobile) or top (on desktop):

- **üìÖ Events** - Create and manage your band crawl events
- **üìç Venues** - Add locations where bands will perform

## Creating Your First Event

### Step 1: Open the Event Wizard

1. Click the orange "Create Event" button at the top
2. You'll see a step-by-step wizard

### Step 2: Enter Event Details

Fill in the required fields:

- **Event Name:** "Long Weekend Band Crawl 2024"
- **Event Date:** The date your event takes place
- **URL Slug:** Short name for the web address (e.g., "vol-6" creates `/events/vol-6`)

**Important:** The slug will be used in the public URL. Keep it short and lowercase.

### Step 3: Add Venues

1. For each venue, enter:
   - Venue name (e.g., "The Analog Cafe")
   - Address (optional but recommended)
2. Click "Add Venue" for each location
3. Click "Next" when done

**Tip:** You can add more venues later if needed!

### Step 4: Add Performances

1. For each band/performer:
   - Band name
   - Select venue from the dropdown
   - Start time (when they go on stage)
   - End time (or set the duration in minutes)
   - Website URL (optional)
2. Click "Add" for each band
3. Repeat for all bands

**Tip:** Add them in chronological order to stay organized!

### Step 5: Review and Publish

1. Review all the information
2. Click "Publish Event"
3. Success! Your event is now visible to the public

**Note:** Your event starts as a draft. Only publish when all information is complete!

## Adding Venues

If you need to add venues to an existing event (or for future events):

### Quick Add Venue

1. Go to the **üìç Venues** tab
2. Click "Add Venue"
3. Fill in the form:
   - **Name** (required)
   - **Address** (optional)
   - **Website** (optional)
   - **Instagram** (optional)
   - **Facebook** (optional)
4. Click "Save"

**Tip:** Add all your venues before scheduling performances!

### Editing a Venue

1. Find the venue in the list
2. Click "Edit"
3. Make your changes
4. Click "Update Venue"

### Viewing Venue History

1. Find the venue in the list
2. Click "History"
3. See all past performances at that venue

**Note:** You cannot delete a venue if bands are currently assigned to it.

## Adding Performances

### Adding a New Performance

1. Go to the **üé∏ Performers** tab
2. Select your event from the dropdown (optional - you can add bands without events)
3. Click "Add Performer"
4. Fill in the form:
   - **Band Name** (required)
   - **Event** (optional - leave blank to add later)
   - **Venue** (optional - can be assigned later)
   - **Start Time**
   - **Duration** OR **End Time**
   - **Website** (optional)
   - **Social Media Links** (optional)
5. Click "Save"

### Editing a Performance

1. Find the performance in the list
2. Click "Edit"
3. Make your changes
4. Click "Update Performance"

### Deleting a Performance

1. Find the performance in the list
2. Click "Delete"
3. Confirm the deletion

**Warning:** Deleting a performance cannot be undone!

### Mobile Tip: Swipe Gestures

On mobile devices, you can swipe left on a band to reveal the delete button quickly.

## Publishing Your Event

### Making Your Event Public

1. Go to the **üìÖ Events** tab
2. Find your draft event
3. Click "Edit"
4. Check the "Published" checkbox
5. Click "Save"

**Important:** Only publish when ALL information is correct! Once published, attendees can see it.

### Unpublishing

Need to make changes to a published event?

1. Go to the **üìÖ Events** tab
2. Find your event
3. Click "Edit"
4. Uncheck "Published"
5. Make your changes
6. Re-publish when ready

## Troubleshooting

### "I can't add a performance - the venue dropdown is empty"

**Solution:** Add venues first! Go to Venues tab and create your venues before adding performances.

### "I see a conflict warning but the times look right"

**Solution:** Make sure you're looking at the full time range. A band ending at 9:00pm conflicts with another starting at 9:00pm. Leave at least 15-30 minutes between acts for setup.

### "My changes aren't showing on the public schedule"

**Solution:**
1. Make sure the event is Published (not Draft)
2. Wait 1-2 minutes for the cache to update
3. Refresh the public page

### "I accidentally deleted a performance"

**Solution:** Unfortunately, deletions are permanent. You'll need to re-add it manually. Double-check before clicking delete!

### "The site isn't loading on my phone"

**Solution:**
1. Try refreshing the page (pull down to refresh)
2. Check your internet connection
3. Try a different browser (Chrome, Safari, Firefox)
4. Make sure you're at the correct URL
5. Contact support if it still doesn't work

### "I can't delete a venue"

**Solution:** You cannot delete a venue that has performances assigned to it. You need to:
1. Go to Performers tab
2. Find all performances at that venue
3. Either delete them or assign them to a different venue
4. Then you can delete the venue

## Tips for Success

### Before the Event

- [ ] Create event at least 2 weeks in advance
- [ ] Add all venues first
- [ ] Add all performances with accurate times
- [ ] Review for conflicts (shown in red)
- [ ] Double-check all times and venues
- [ ] Test the public schedule
- [ ] Publish event

### During Setup

- [ ] Keep venue names consistent (don't use "The Analog" and "Analog Cafe")
- [ ] Use 24-hour time for clarity, or include AM/PM
- [ ] Add 15-30 minute buffers between bands for setup
- [ ] Test the public schedule on multiple devices
- [ ] Get feedback from someone else

### Best Practices

- **Save Often** - Don't rely on auto-save
- **Double-Check Times** - Incorrect times cause attendee confusion
- **Keep It Simple** - Don't overcomplicate band names or venues
- **Test Before Publishing** - Always preview before going live
- **Backup Important Info** - Screenshot your schedule
